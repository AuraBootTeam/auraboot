「高级·原子动作逃生舱」= 旧矩阵在 v2 的去处（保留，不删）
  • 默认折叠，普通管理员根本不展开——日常只勾①业务能力（飞书体验）。
  • 展开后 = 现在的 PermissionMatrix 的能力，但不是密集网格：改成"权限码 + 名称 + 数据范围 + 来源"的可搜索分组表。
  • 「来源」列是关键：绝大多数原子码是被上面某个能力授予的（绿色），在这里只是审计核对；少数没被任何能力覆盖的（黄色）才是真正在这里破例授予的对象。
  • 给需要审计/破例的 ~5% 用户。任何"破例"会计入②有效权限的"破例"计数，留痕可审。
三维各自独立（别混编）
