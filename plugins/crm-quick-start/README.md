# CRM Quick Start Template

Lightweight CRM for small teams. One-click install to get a complete lead-to-deal pipeline.

## What's Included

| Model | Description |
|-------|-------------|
| Lead | Track prospects from capture through qualification |
| Account | Companies and organizations |
| Contact | People at customer companies |
| Opportunity | Sales deals with pipeline stage tracking |

## Key Features

- **Lead Lifecycle**: NEW → CONTACTED → QUALIFIED → LOST
- **Sales Pipeline**: PROSPECTING → PROPOSAL → NEGOTIATION → CLOSED_WON/LOST
- **Account 360**: View contacts and opportunities linked to each account
- **Named Queries**: Pipeline summary and lead conversion stats for dashboards

## Installation

Import via Plugin Manager or CLI:

```bash
aura plugin import plugins/crm-quick-start
```

## Namespace

All resources use the `tcrm` namespace prefix.
