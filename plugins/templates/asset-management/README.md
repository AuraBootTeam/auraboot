# Asset Management Template

IT and office asset tracking with maintenance records.

## What's Included

| Model | Description |
|-------|-------------|
| Asset | Physical/software assets with lifecycle tracking |
| Category | Asset classification for reporting |
| Maintenance | Repair, inspection, and upgrade records |

## Key Features

- **Asset Lifecycle**: AVAILABLE → IN_USE → UNDER_MAINTENANCE → RETIRED
- **Assign/Return**: Track who is using each asset
- **Maintenance Tracking**: Schedule and complete maintenance tasks
- **Asset 360**: View maintenance history per asset
- **Named Query**: Asset value summary by category

## Installation

```bash
aura plugin import plugins/templates/asset-management
```

## Namespace

All resources use the `tasset` namespace prefix.
