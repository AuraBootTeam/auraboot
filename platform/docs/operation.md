## @^2的含义
- @ : 版本分隔符，表示引用特定版本的schema
- ^2 : 兼容版本范围，表示"兼容2.x.x版本"
    - 允许使用 2.0.0 、 2.1.0 、 2.9.9 等
    - 不允许 使用 3.0.0 （主版本变更可能有破坏性改动）
    - 不允许 使用 1.x.x （版本过低）
## 版本范围规则
- ^2.1.0 : 兼容 >=2.1.0 <3.0.0
- ^2 : 等同于 ^2.0.0
- ~2.1.0 : 兼容 >=2.1.0 <2.2.0 （更严格）
- 2.1.0 : 精确版本匹配
