# AuraBoot Product Overview

AuraBoot is a low-code/no-code platform framework that enables dynamic form and page generation through JSON schema definitions. It's a meta-driven application framework built on Spring Boot.

## Core permissions

- **Dynamic Schema Management**: Define entities, fields, and relationships through metadata rather than code
- **Form & List Generation**: Automatically generate CRUD interfaces from JSON schema definitions
- **Page Designer**: Visual page builder with component registry and publishing workflow
- **Multi-tenant Architecture**: Built-in tenant isolation and management
- **RBAC System**: Role-based access control with permission management
- **File Management**: File upload and relationship tracking
- **Event System**: Local and distributed event publishing/listening
- **Internationalization**: Multi-language support with i18n configuration

## Key Concepts

- **Entity Definition**: Metadata-driven entity and field definitions stored in database
- **Page Schema**: JSON-based page layout and component configuration
- **Component Registry**: Reusable UI components with schema definitions
- **Service Orchestration**: Configurable business logic workflows defined in YAML
- **Dynamic Controller**: Generic REST endpoints that operate on metadata definitions

## Target Use Cases

Building admin panels, internal tools, and data management systems without writing repetitive CRUD code. The platform allows developers to define data models and UI schemas declaratively, then automatically generates the necessary APIs and interfaces.
